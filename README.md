# Sirah Digital — Multi-Branch Business Operating System

A landing page for the Sirah Digital multi-branch business operating system.

## Tech Stack

- Vite
- TypeScript
- React
- Tailwind CSS
- Framer Motion
- Radix UI (shadcn-ui)

## Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
